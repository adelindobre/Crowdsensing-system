register_thread(self, node, thread)

source code 

Registers the given thread as belonging to the given node. The node must register threads with its own datastore. The threads will then be able to query the datastore. Multiple threads (logically owned by 'node') can be registred to the datastore.

Parameters:
  • node (Node) - the node owning the datastore
  • thread (Thread) - the thread to register

get_A(self, node, column)

source code 

Returns an element from the row of the A matrix that is stored in this datastore. This is a blocking operation. The maximum number of in-flight requests is limited, see get_max_pending_requests().

Parameters:
  • node (Node) - the node accessing the datastore; must be the node that owns the datastore
  • column (Integer) - the column of the element
Returns: Float
the element of matrix A at the requested position

Updates an element from the row of the A matrix that is stored in this datastore. This is a blocking operation. The maximum number of in-flight requests is limited, see get_max_pending_requests().

  • column (Integer) - the column of the element
