Class TestGenerator

source code

Randomly generates tests, tests are composed of a matrix, a number of nodes and a number of tasks.

For loading a test's parameters from a file with the following format:

param_name1 = value

param_name2 = value

[...]

Blank lines or lines starting with # (comments) are ignored

Parameter names are defined in this class. Parameters can be declared in any order in the file.
