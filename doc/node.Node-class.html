Parameters:
  • nodes (List of Node) - a list containing all the nodes in the cluster

schedule_task(self, task, in_slices, out_slices)

source code 

Schedule task to execute on the node.

Parameters:
  • task (Task) - the task object to execute
  • in_slices (List of (Integer, Integer, Integer)) - a list of the slices of data that need to be gathered for the task; each tuple specifies the id of a node together with the starting and ending indexes of the slice; the ending index is exclusive
  • out_slices (List of (Integer, Integer, Integer)) - a list of slices where the data produced by the task needs to be scattered; each tuple specifies the id of a node together with the starting and ending indexes of the slice; the ending index is exclusive

shutdown(self)

source code 

Instructs the node to shutdown (terminate all threads). This method is invoked by the tester. This method must block until all the threads started by this node terminate.
